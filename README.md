# MasterTCUIMP
El proyecto consta de los siguientes ficheros:
* gen_script.py: Es el script que contiene el código que genera los datasets aleatorios y representa gráficamente las siguientes funciones de distribución: Normal (Gausiana), Uniforme y Beta.
* compare-distributions.png: Es la imagen de salida generada por el script de Python, que contiene las gráficas de las distribuciones generadas de forma aleatoria junto con la gráfica de comparación del conjunto.
* homework_ud_1.ipynb: Es el notebook que contiene la solución del ejercicio de la UD1.
